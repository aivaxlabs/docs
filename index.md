# AIVAX Documentation

AIVAX is an AI orchestration platform for building assistants on top of hosted or BYOK language models, RAG collections, tools, skills, workers, and chat channels.

Use these docs to:

- Make your first OpenAI-compatible inference call.
- Create and protect API keys.
- Understand account balance, plan limits, and usage billing.
- Connect gateways, collections, tools, skills, chat clients, and batch workflows.
